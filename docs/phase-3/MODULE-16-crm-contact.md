# Module 16: CRM & Contact Management

## Status: [ ] Not Started
## Category: ERP
## Phase: 3 — Platform Maturity
## Priority: P3 - Medium

---

## Overview

A lightweight manufacturing-focused CRM that manages customers, contacts, sales
opportunities, and communication history. Includes a customer self-service portal
where customers can submit RFQs, track order status, download certs, and approve
first articles — a major competitive differentiator vs. ProShop.

**ProShop Improvements**: Sales pipeline management, activity logging, customer
communication history, win/loss analysis, automated touchpoints (quote follow-ups,
order confirmations, shipping notifications), and the customer portal for order
self-service.

```csharp
public class Customer
{

    public string CustomerCode { get; set; } = string.Empty;   // Short code: "BOEING"
    public string CompanyName { get; set; } = string.Empty;
    public string? Industry { get; set; }                       // Aerospace, Medical, Defense
    public string? Website { get; set; }
    public string? BillingAddress { get; set; }
    public string? ShippingAddress { get; set; }
    public string? PaymentTerms { get; set; }
    public CustomerStatus Status { get; set; } = CustomerStatus.Active;
    public string? Notes { get; set; }
    public string? SalesOwnerId { get; set; }                  // Assigned sales person
    public DateTime CreatedAt { get; set; } = DateTime.UtcNow;
    public ICollection<Contact> Contacts { get; set; } = new List<Contact>();
```

### Step 4 — Create Opportunity Model

```csharp
public class Opportunity
{
    public int Id { get; set; }
    public int CustomerId { get; set; }
    public Customer Customer { get; set; } = null!;
    public string Title { get; set; } = string.Empty;          // "Annual titanium housings"

    public decimal EstimatedValue { get; set; }
    public OpportunityStage Stage { get; set; } = OpportunityStage.Prospecting;
    public int ProbabilityPct { get; set; } = 50;
    public DateTime? ExpectedCloseDate { get; set; }
    public int? LinkedQuoteId { get; set; }
    public Quote? LinkedQuote { get; set; }
    public string? LostReason { get; set; }                    // If lost: Price, Timeline, Competition
    public string OwnerId { get; set; } = string.Empty;
    public DateTime CreatedAt { get; set; } = DateTime.UtcNow;
    public DateTime? ClosedAt { get; set; }
}

public enum OpportunityStage
{
    Prospecting, Qualification, QuoteRequested, QuoteSent, Negotiation, Won, Lost
}
```

### Step 11 — Customer Portal
**New Files**: `Components/Pages/Portal/` directory

**Portal Login**: `/portal/login` — separate portal authentication (contacts only)
**Portal Dashboard**: `/portal/dashboard` — customer sees their orders and quotes
**Portal Order Status**: `/portal/orders` — order list with stage visibility
**Portal Certs**: `/portal/certs` — download inspection reports and certs

Portal authentication: separate from internal auth — use `Contact.HasPortalAccess` and `PortalPasswordHash` with a separate cookie scheme.

**Configure in Program.cs**:
```csharp
builder.Services.AddAuthentication()
    .AddCookie("PortalScheme", options =>
    {
        options.LoginPath = "/portal/login";
        options.ExpireTimeSpan = TimeSpan.FromHours(8);
    });
```

Portal pages should have `[Authorize(AuthenticationSchemes = "PortalScheme")]` requirement.
